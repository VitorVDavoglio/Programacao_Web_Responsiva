Praticando HTML

Alguns caracteres não podem ser mostrados na página sem o uso de códigos pois são baseados para o HTML interpretar

Os cógidos em HTML tem a seguinte forma: &cógido

temos códigos por nome: &nome ou por números: &#número

Primeiro é para colocar mais espaços entre as palavras. Posso dar mais espaços na programação assim: " " mas não funciona então é necessário usar o código:"& nbsp" por exemplo: as     sim

-ESPAÇO:"& nbsp" ou & #160 -MENOR QUE:"& lt" ou "& #60" -MAIOR QUE:"& gt" ou "& #62"; -E COMMERCIAL:"& amp" ou "& #38" -COPYRIGHT:"& copy" ou "& #169" -TRADE MARKER:"& trade" ou "& #8482" -MARCA REGISTRADA:"& reg" ou "& #174" -ASPAS:"& quot" ou "& #34" -TRAVESSÃO:"& mdash" ou "& #8212" -COLCHETE ESQUERDO:"& lbrack" ou "& #91" -COLCHETE DIREITO:"& brack" ou "& #93" -CHAVE ESQUERDA:"& lbrace" ou "& #123" -CHAVE DIREITA:"& rbrace" ou "& #125"

Para colocar imagens pelo HTML usamos a TAG "<img>" dentro do img colocasse "src=" para localizar a imagem, serve tanto para imagens local quanto URL

O atributo "alt" é inserido do lado da "src" para utilizar o leitor de tela caso a imagem não possa ser carregada

Para alterar a largura="width"e a altura="height" da imagem, utiliza-se porcentagem ou pixels

A inserção de LINKS se usa a TAG "<a>" e dentro coloca o atributo "href" tanto para links externos ou para outras partes da página, para que abra uma nova aba ao clicar no link deve-se colocar o atributo=>target="blank"

EXEMPLO:<a href="link">Acesse o link aqui</a>

Para ligar partes da própria página atual é necessário colocar o atributo=>id="paragrafo1" dentro da TAG "". Para estabelcer um link para o parágrafo destacado usa o atributo=>href="#paragrafo1" dentro da TAG <a> vá para parágrafo1 </a>

Para transformar uma imagem em um link basta colocar uma imagem (<img>) entre as tags <a>

Parágrafo centralizado

Atributo "bgcolor" troca a cor de fundo da página

Atributo "text" troca a cor do texto da página

Atributo "link" troca a cor do link da página

Atributo "vlink" troca a cor do link visitados da página

Dentro da TAG "body" pode trocar a imagem de fundo da página com o atributo "background"

A TAG "font" podendo utilizar os atributos como:"size" para tamanho da fonte, "face" para o tipo de fonte, "color" para o cor da escrita
