A linguagem HTML

A linguagem HTML (Hypertext Markup Language) é a linguagem que utilizamos para montar a estrutura básica das páginas web.Trata-se de
uma linguagem de difícil fácil aprendizagem. Para fixar suas construções e necessário praticar.

Praticando nossas primeiras tags

Com ajuda das tags aprendidas na aula de hoje, podemos escrever fórmulas como CO2 e CH4, assim como unidades de medida como m2 e cm3.

Já podemos até escrever poemas:
Minha terra tem palmeiras,
Onde canta o Sabiá;
As aves que aqui gorjeiam,
Não gorjeiam como lá.

Algumas coisas que aprendemos nessa aula:

  • A estrutura básica de um documento HTML
  • Tags de formatação de texto
  • Criarmos listas de HTML
