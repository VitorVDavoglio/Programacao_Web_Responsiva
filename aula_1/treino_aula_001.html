Segundo parágrafo, essa separação é feita usando a TAG "p"

Quando estamos escrevendo e queremos quebrar a linha,
a TAG "br" entra em ação.

Ao adicionarmos a TAG "hr" fazermos a separação da página com uma linha

Podemos colocar palavras em negrito usando as Tags "strong" ou "b" strong ou b;

Ou em itálico com as Tags "em" ou "i" por exemplo Frango ou whey;

TAG "sub" em caso de números inferiores como: H2O ou C3;

TAG "sup" em caso de números superiores como: 2x4= 8 como 23;

Vamos destacar uma frase usando a TAG "u" sublinhando: A utilização do cigarro é prejudical a saúde;

Podemos simular um erro de colocação= TAG "s" ou "strike" dessa form forma;

Agora utilizaremos as listas:

Lista não ordenada usamos a TAG "ul" e cada elemento começa com a TAG "li"

  • Whey
  • Creatina
  • Ômega 3

Agora a TAG "ol" faz a lista ter uma ordem por exemplo:

  1. Whey
  2. Creatina
  3. Ômega 3

Dentro da TAG ol podemos mudar o tipo do marcador pois temos alguns tipos:

Tipo=1 é a numeração arábica
  1. Whey
  2. Creatina
  3. Ômega 3
Tipo=I é a numeração romana
  1. Whey
  2. Creatina
  3. Ômega 3
Tipo=i é a numeração romana minúscula
  1. Whey
  2. Creatina
  3. Ômega 3
Tipo=A é a ordenação com letra maiúscula
  1. Whey
  2. Creatina
  3. Ômega 3
Tipo=a é a ordenação com letra minúscula
  1. Whey
  2. Creatina
  3. Ômega 3

Dentro da lista também é possível mudar a numeração inicial usando a TAG "start"

Start=6
  1. Whey
  2. Creatina
  3. Ômega 3

Agora a lista de descrição para utilizar é preciso colocar a TAG "dl" e as outras são:
TAG "dl" para cada termo a ser descrito
TAG "dd" para a descrição de cada termo, por exemplo:

Alcatra
É uma carne que vem do boi.
Sasami
pacote feito com filés de peito de frango
